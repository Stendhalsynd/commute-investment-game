# AGENT

## 작업 로그 규칙 (mandatory)
- 모든 작업 완료 또는 중단 시점마다 `HANDOFF.md`에 기록한다.
- 기록은 최신 순으로 맨 위에 추가한다.
- 각 항목은 아래 항목을 최소 포함한다.
  - 실행 작업 요약(무엇을, 왜 했는지)
  - 변경 파일 목록
  - 현재 상태(완료/진행/차단)
  - 다음 액션 또는 리스크

## `HANDOFF.md` 기록 형식(권장)
- 제목: 날짜 또는 세션 단위 섹션
- 하위 항목: 진행 요약, 이번 작업 산출물, 남은 항목, 참고 상태

## 예외/주의
- `HANDOFF.md`는 추적용 문서이며 기능 코드와 동일한 신뢰도 기준으로만 참조한다.
- 수동 QA는 완료 후 해당 결과를 `QA_RESULTS.md`에 반영한다.
